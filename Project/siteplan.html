Testing (self-check)

Run these checks as you develop the project:

  • HTML validation (W3C validator) — ensure no structural errors.
  • Accessibility (use Lighthouse & axe) — headings, alt text, ARIA where needed.
  • Contrast checks — confirm text meets AA/AAA where required.
  • Performance — validate images, minimize blocking CSS/JS, test with Lighthouse/PageSpeed.
  • SEO basics — page titles, meta description, semantic headings.

Document any issues you find and the fixes you applied so your submission notes show testing work.
